Propriedade box-sizing

Define como é calculada a lardura e a altura de um elemento, tenham eles padding e border incluso, ou não.

box-sizing: content-box (padrão):

A largura é aplicada apenas ao conteúdo do elemento:

Sendo assim, soma-se à largura os parâmetros de padding e border estabelecidos.

    
    #exemplo1 {
        box-sizing: content-box;  
        width: 300px;
        padding: 30px;  
        border: 10px solid blue;
    }

Exemplo 1

Essa div tem largura de 300px.

Mas a largura total é de 300px + 20px (de borda esquerda e direira) + 60px (de padding na direira e na esquerda) = 380px!

box-sizing: border-box:

A largura é aplicada a todas as partes do elemento: conteúdo, padding e bordas:

    
    #exemplo2 {
        box-sizing: border-box;
        width: 300px;
        height: 300px;
        padding: 30px;  
        border: 10px solid blue;
    }

Exemplo 2

Aqui, a largura total é 300px, independente de qualquer coisa!

Box-sizing universal

Uma vez que o resultado do uso do "box-sizing: border-box" é muito melhor, muitos desenvolvedores querem que todos os elementos em suas páginas operem dessa forma.

O primeiro estilo no head assegura que todos os elementos estão dimensionados de forma mais intuitiva. 

    
* {
    box-sizing: border-box;
}

#exemplo3 {
    width: 300px;
    height: 300px;
    padding: 40px;
    border: 5px solid blue;
}

#exemplo4 {
    width: 300px;
    height: 300px;
    padding: 10px;
    border: 5px solid red;
}

Exemplo 3

Essa div tem largura total de 300px, independente de padding e/ou border!

Exemplo 4

Essa div também tem largura total de 300px, independente de padding e/ou border!
